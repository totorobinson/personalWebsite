The Package Protector


The objective for our Senior Project was to address the issue of package theft. As a team, we worked towards developing a product that would meet specific design requirements and provide a solution to this issue. Our hard work paid off as our project won 1st place in our graduating class.

Design Requirements:
1. Prevent unauthorized access and securely hold packages
2. Keep packages intact and undamaged
3. Indicate when a package is ready for pickup
4. Detect and deter tampering
5. Provide easy user access
6. Support packages up to 15lbs
7. Withstand light rain and snow

We developed the Package Protector to meet these specifications. It features a lift mechanism that safely lowers packages into the box, a locking mechanism that restricts access to authorized users only, an alarm system that detects and deters tampering, and a mobile app that allows users to interact with the box seamlessly.

As a team, we recognized the importance of collaborating effectively and efficiently to achieve our goals. Therefore, we split up the workload between teammates, based on each person's strengths and expertise, and integrated our individual efforts to create a cohesive product. To ensure that we completed the project on time, we adopted an Agile workflow approach. We had frequent standup meetings where we discussed our progress, identified roadblocks, and devised solutions to overcome them. These meetings allowed us to stay aligned, adjust our priorities as necessary, and address any issues that arose promptly. When one team member encountered a challenge, we collaborated to find a solution and ensure that everyone remained on track.

During the project, my primary responsibilities were to design and integrate the android mobile app with the package protector, develop the embedded system code that controlled the scissor lift, and design and build the lift itself. My focus on the mobile app was to create an intuitive and user-friendly interface that communicated with a raspberry pi through AWS cloud services. Through the app, users could easily push buttons to signal the raspberry pi to open the package retrieval door or turn off the alarm. Conversely, the raspberry pi could signal the mobile device, alerting the user when packages had arrived or if an intruder had been detected.

Tools
Java (Android), Python (Raspberry Pi), C (Embedded System), AWS
