NLP Question Answering System

As a final project for our Natural Language Processing course, my partner and I developed a rule-based Question Answering program. This endeavor allowed us to leverage various skills and techniques we acquired throughout the course. Our program parsed a given passage and generated accurate answers to multiple questions derived from the text. Our source material was the Canadian Broadcasting Corporation web page for kids, which provided a fantastic dataset to work with.

The algorithm we devised is as follows:
1. Pre-process the story and question text: This entailed tokenizing all sentences using stopword removal, punctuation removal, and lemmatization.

2. Passage retrieval and ranking using BM25: We queried the tokenized story with each sanitized question to retrieve the top 3 sentences that likely contain the answer.
Example showing original question, query words from question, and top three sentences from passage:
3. Question typing (Who, Where, When, etc): We did Named Entity Recognition based on the type of question. For example, if the question began with “Who”, we would search the top three sentences for instances of people or organizations.
Example showing NER:
4. Answer identification: We would identify the first returned entity to be the answer to the question. Because our sentences were already ranked in order of most likely containing the answer, returning the first answer proved to be the best method.

Our system was a success, placing us in the top three teams out of 60 based on performance. Our final product achieved scores: Recall: 0.5330, Precision: 0.4152, F-measure: 0.4668.

The project was both challenging and enjoyable. We experimented with various approaches, including question expansion with WordNet, exploring adjacent sentences for answers, and employing dependency parsing, but these attempts did not improve our performance. This project also highlighted the complexity of natural language processing and how it remains a challenging field. It really highlighted the importance of machine learning, and if we were to undertake the project again, we would explore using a machine learning model to evaluate whether it would enhance our results by utilizing an entirely different approach.

Tools Used
Python, SpaCy, BM25
