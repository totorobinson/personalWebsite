An Educational Game



Our team developed a dynamic and engaging C++ desktop application using the Qt framework. Our focus was on educating young children about the principles of color theory in a fun and interactive way. The application offers interactive demonstrations, using various color combinations, that showcase how colors interact with each other. Children can experiment with primary, secondary, and tertiary colors, and observe the results of their combinations.

In addition to these interactive demos, our application also features a built-in game, which employs Box2d, a popular physics engine. The game is designed to challenge children's understanding of color theory by requiring them to use the knowledge of what they have just learned to win the game.

To enhance the user experience, we implemented an intuitive user interface, featuring vibrant graphics and a simple navigation system. Our app is easy to use and accessible to children of all ages and skill levels. We are proud to have created a valuable tool for educators and parents alike, providing children with a fun and interactive way to learn about color theory.

Tools
C++, Qt, Box2d
